# DLL Injector

Это приложение на C++ предназначено для инъекции DLL-файлов в запущенные процессы Windows. Оно позволяет вводить имя процесса и имя DLL, которую необходимо инъектировать, а затем выполняет процедуру инъекции.

## Установка и сборка

### Необходимые зависимости

Для сборки и выполнения этого приложения необходимо иметь установленный компилятор C++ (например, Visual Studio) и библиотеку Windows API.

### Шаги по сборке

1. Откройте injection.exe

## Использование

1. Запустите приложение.
2. Когда будет предложено, введите имя процесса, в который хотите инъектировать DLL (например, `stalcraft.exe`).
3. Затем введите путь к DLL, которую хотите инъектировать (например, `dorian_client.dll`).

Приложение получит идентификатор процесса (PID) указанной программы и выполнит инъекцию DLL в этот процесс.

### Примеры использования

```
Launching the program...
Write name DLL (for example, stalcraft.exe): stalcraft.exe
A process with a PID was found: 1234
Write name DLL (for example, dorian_client.dll): dorian_client.dll
C:\Path\To\dorian_client.dll
Completion.
```

## Примечания

- Приложение требует повышенных прав для инъекции DLL, поэтому его необходимо запускать с правами администратора.
- Убедитесь, что указанный процесс запущен перед использованием.
- Будьте осторожны с инъекцией DLL в чужие процессы, так как это может привести к неправильной работе программ или нарушению безопасности.
